Set different color values

Color: "Tomato":

rgb(255, 99, 71)

#ff6347

hsl(9, 100%, 64%)

Mismo color pero con trasparencia 50%:

rgba(255, 99, 71, 0.5)

hsla(9, 100%, 64%, 0.5)

Además de los nombres de colores predefinidos, los colores se pueden especificar usando RGB, HEX, HSL o incluso colores transparentes usando valores de color RGBA o HSLA.
