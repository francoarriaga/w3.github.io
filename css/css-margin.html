Automatic margin

Puede configurar la propiedad de margen en automático para centrar horizontalmente el elemento dentro de su contenedor. El elemento ocupará entonces el ancho especificado y el espacio restante se dividirá en partes iguales entre los márgenes izquierdo y derecho:

This div will be horizontally centered because it has margin: auto;

Inherited margin

Deje que el margen izquierdo se herede del elemento padre:
